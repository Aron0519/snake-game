# Snake Game (Python + Pygame)

A simple classic Snake game built using Python and Pygame.

## Features
- Classic snake gameplay  
- Score tracking  
- Keyboard controls  
- Restart support  

## Controls
- Arrow keys: Move the snake  
- Close window: Quit the game  

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Aron0519/snake-game.git
   cd snake-game

2. Install dependencies:
   pip install -r requirements.txt

3. Run the game:
   python snake.py

Tech Stack
Python, Pygame

Notes
If the game window does not appear when running from VS Code on macOS, try running it from the system Terminal instead.
